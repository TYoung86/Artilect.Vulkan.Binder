Remarks
Valid Usage
  • If aspectMask contains VK_IMAGE_ASPECT_COLOR_BIT, it must not contain either of VK_IMAGE_ASPECT_DEPTH_BIT or VK_IMAGE_ASPECT_STENCIL_BIT
  • aspectMask must not contain VK_IMAGE_ASPECT_METADATA_BIT
  • mipLevel must be less than the mipLevels specified in IVkImageCreateInfo when the image was created
  • (baseArrayLayer + layerCount) must be less than or equal to the arrayLayers specified in IVkImageCreateInfo when the image was created
Valid Usage (Implicit)
  • aspectMask must be a valid combination of VkImageAspectFlags values
  • aspectMask must not be 0
